# 正则表达式

## 元字符

| 字符 | 描述                                                                   |
| :--- | :--------------------------------------------------------------------- |
| \s   | 匹配任何空白字符，包括空格、制表符、换页符等等。等价于 [ \f\n\r\t\v]。 |

## 应用

### 匹配数字，保留小数后 2 位

```bash
/^\d+(\.\d{0,2})?$/
```

### 去除\*号后面所有数字

```bash
\\*.*?$
```
