# kafka 安装

## kafka安葬

首先，安装kafaka之前需要安装zookeeper
## kafaka 测试

创建topic
```bash
bin/kafka-topics.sh --create --zookeeper localhost:2181 --replication-factor 1 --partitions 1 --topic test
```

生产者启动

```bash
bin/kafka-console-producer.sh --broker-list 127.0.0.1:9092 --topic test
```

消费者启动

```bash
bin/kafka-console-producer.sh --broker-list 127.0.0.1:9092 --topic test
```
